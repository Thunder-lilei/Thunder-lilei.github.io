Vue学习（基础练习）
2020-12-17
前端 js vue
technology

作者 ： 我有一把小榔头丶

CDN

  • Staticfile CDN（国内） : https://cdn.staticfile.org/vue/2.2.2/vue.min.js

  • unpkg：https://unpkg.com/vue/dist/vue.js, 会保持和 npm 发布的最新的版本一致。

  • cdnjs : https://cdnjs.cloudflare.com/ajax/libs/vue/2.1.8/vue.min.js

helloword

每个 Vue 应用都需要通过实例化 Vue 来实现。

特点

  1. JavaScript框架
  2. 简化Dom操作
  3. 响应式数据驱动

引入

EL挂载点

data

Vue中需要的数据定义在data中
遵守js语法

Vue指令

  • v-开头的特殊语法

  1. v-test

    设置标签内容

    1. 内容会被替换
    2. 使用大括号形式不会被替换
    3. 可以使用''进行拼接

  2. v-html

    解析html结构

  3. v-on

    绑定事件
    可以简写为@
    绑定的方法写在method中
    通过this获取data中定义的数据

     ![Snipaste_2020-12-17_14-36-40.png](https://i.loli.net/2020/12/17/4PsT6o2u5ckEMCJ.png)
 函数调用携参
 可以通过事件修饰符对事件进行限制
 .enter限制触发为回车

  4. 计数器案例

  5. v-show

    元素的显示和隐藏，响应式
    根据表达式的真假切换

  6. v-if

    和v-show的区别：

    • v-if 操纵dom
    • v-show操纵样式，切换消耗少

  7. v-bind

    1. 对元素属性的操作
    2. 动态参数
      <a v-bind:[attributeName]="url"> ... </a>
      根据Vue实例data中的attributeName变量赋值

  8. v-for

    遍历数据
    index item 序号 对象

  9. v-model

    获取和设置表单元素的值（双向数据绑定）

  10. 记事本

    实现增加、删除、统计、清空、隐藏
