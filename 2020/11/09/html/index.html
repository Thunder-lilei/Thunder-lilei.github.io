作者 ： 我有一把小榔头丶

介绍

  • 超文本标记语言
  • 标记语言是一套标记标签（用来描述网页）
  • .htm或.html
  • 大小写不敏感

html标签（标记）

  • 尖括号包围<html>
  • 通常成对出现<b></b>，第一个叫开始标签（开放标签），第二个叫结束标签（闭合标签）
  • 空标签，只包含属性，没有闭合标签

  1. <h1>—<h6>标题

  2. <p>段落

    • 浏览器会自动在段落前后添加空行
    • 应该使用<br/>替换<p></p>来实现空行

  3. <a>链接

    • href属性指定链接地址
    • target="_blank"，链接会在新窗口打开

  4. <img>图像

    • src图片地址
    • width，height图片尺寸
    • alt，替换文本属性，图像不加载时显示的文本

  5. <hr>水平线

    • 分割内容

  6. <!-- -->注释

    • 提高可读性
    • 浏览器会忽视注释

  7. <br/>折行

    • 不产生新段落的情况下换行
    • 浏览器会移除源代码众多与的空格和换行，所有连续的空格和换行会被算作一个空格

  8. <table>

    • 包含行<tr>，单元格<td>，表头<th>（大多数浏览器为粗体居中）
    • border定义边框，通常情况下无边框
    • 空的单元格可以使用空格占位符将边框显现出来
    • caption，定义表格标题，居中唯一
    • thead，页眉，tbody，主题，tfoot，页脚

  9. <li></li>列表项

    • 无序列表

      <ul></ul>
      可以嵌套使用，标记点会有变化

    • 有序列表

      • type递增标点类型
        <ol></ol>

  10. <div>和<span>块

    1. <div>是块级元素
      • 表格布局
      • 对内容块设置样式属性
    2. <span>是内联元素
      • 为部分文本设置样式属性

html文档 = 网页

  • html文档描述网页
  • 文档包含html标签和纯文本
  • 浏览器不会显示html标签，而是使用标签解释页面内容
    例：
    <html>文本</html>，文本描述网页
    <body>文本</body>，文本可见的网页内容

html元素

  • html文档由html元素定义
  • 从开始标签到结束标签的所有代码
  • 可以有空内容（在开始标签中进行关闭）
  • 大多数元素可以有属性
  • 大多数可以嵌套
  • 大多数html元素分为块级元素（通常以新行开始结束）和内联元素（通常不以新行开始）

html属性

  • 名称/值
  • 开始标签中规定
  • 属性值加引号""

  1. 避免使用的标签和属性

    • 应使用样式替代
    • 标签
      • <center>，定义居中
      • font/basefont，定义字体
      • s/strike，定义删除线文本
      • u，定义下划线文本
    • 属性
      • align，定义文本对齐方式
      • bgcolor，定义背景颜色
      • color，定义文本颜色

  2. href，<a>中指定，链接的地址

  3. algin，对齐方式

    • center，居中
    • left，right
    • 文本
      • text-align:对齐方式，文本对齐

  4. border，边框

  5. class，元素类名

  6. id，唯一

  7. style，行内样式

    • 改变所有html元素的样式的通用方法
    • 也可以在独立的样式表（css文件）中定义
    • background-color:颜色，定义背景颜色
    • 字体
      • font-family:字体系列名称，字体系列
      • font-size:大小，字体大小
      • color:颜色，字体颜色

  8. title，规定元素额外信息

  9. name

    • 对读者不可见
    • 可以创建直接跳至改命名锚的链接

  • 相对路径

html头部元素

  • <title>文档标题

    • 是必需的
    • 页面被添加到收藏夹时显示的标题
    • 搜索引擎结果中的页面标题
  • 有target="_blank"或没有链接都会在新表签页中打开

  • <meta>文档关键词

    • 规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据
    • 一些搜索引擎会利用 meta 元素的 name 和 content 属性来索引您的页面
      例：<meta name="keywords" content="HTML, DHTML, CSS, XML, XHTML, JavaScript, VBScript">
    • 重定向
      例：<meta http-equiv="Refresh" content="5;url=http://www.w3school" />

  • <base>

    • 为页面上所有链接提供默认地址
      例：<base href="http://www.w3school.com.cn/images/" />

  • <link>

    • 定义文档与外部资源之间关系
    • 连接样式表
      例：<link rel="stylesheet" type="text/css" href="mystyle.css" />

  • <style>

    • 定义样式

  • <script>

    • 脚本

html统一资源定位符（url）

语法规则

scheme://host.domain:port/path/filename

  • scheme - 定义因特网服务的类型。最常见的类型是 http
  • host - 定义域主机（http 的默认主机是 www）
  • domain - 定义因特网域名，比如 w3school.com.cn
  • :port - 定义主机上的端口号（http 的默认端口号是 80）
  • path - 定义服务器上的路径（如果省略，则文档必须位于网站的根目录中）。
  • filename - 定义文档/资源的名称
